498. Diagonal Traverse

2月 19, 2016 发布在 LeetCode

Given a matrix of M x N elements (M rows, N columns), return all elements of the
matrix in diagonal order as shown in the below image.

Example:

题意：

给一个M*N的矩阵，按照图中的对角线顺序返回所有元素

思路：

二维数组，注意点在边界的处理

对角线运行分两个方向，右上和左下;

在不考虑边界遍历时两个运动方向的偏移量右上(-1, 1), 左下(1, -1)

在遍历中遇到边界的情况的偏移量:

  1. 右上方向 && 遇到上边界:
    • 遇到右边界: (0, 1)
    • 未遇到右边界: (1. 0)
  2. 左下方向 && 遇到左边界:
    • 遇到下边界: (1, 0)
    • 未遇到下边界: (0, 1)
